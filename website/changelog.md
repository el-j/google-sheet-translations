# Changelog

This changelog is automatically generated by [semantic-release](https://semantic-release.gitbook.io/) from [Conventional Commits](https://www.conventionalcommits.org/).

Every commit to `main` that contains a releasable change type automatically:
1. Bumps the version in `package.json`
2. Updates this changelog
3. Creates a GitHub Release
4. Publishes to npm

> 💡 **Tip:** Run `npm run changelog:preview` locally to see what the next release entry will look like before pushing commits.
