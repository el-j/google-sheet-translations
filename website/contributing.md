# Contributing

See the [CONTRIBUTING.md](https://github.com/el-j/google-sheet-translations/blob/main/CONTRIBUTING.md) file in the repository for the full guide.

## Quick reference — Conventional Commits

| Prefix | Version bump | Example |
|--------|-------------|---------|
| `feat:` | minor | `feat(sync): add retry logic` |
| `fix:` | patch | `fix(writer): sanitize locale names` |
| `perf:` | patch | `perf: cache header row` |
| `refactor:` | patch | `refactor: extract column helper` |
| `feat!:` or `BREAKING CHANGE:` | **major** | `feat!: rename main function` |
| `docs:`, `style:`, `test:`, `ci:`, `chore:` | none | `docs: update README` |
